# SkyGuide SKSE

TypeScript Skyrim Platform plugin for game state extraction.

## Prerequisites

- Skyrim Special Edition
- Skyrim Platform (Nexus #54909)
- Node.js 20+
- pnpm

## Install

```bash
pnpm install
```

## Build

```bash
pnpm build
```

## Dev

```bash
pnpm dev
```

Runs webpack in watch mode.

## Deploy

Copy `dist/skyguide.js` to `Data/Platform/Plugins/` in your Skyrim folder.
